遇到的一些问题及其解决方案

Q1：遇到Vscode的scanf报错问题

这个问题有两种解决方案：

第一种是比较懒省事的方案，就是在项目属性里添加【/D _CRT_SECURE_NO_WARNINGS】，让他不再报错

第二种就是使用vscode推荐的安全函数scanf_s，这个函数还有一些类似的，比如fopen_s等等，他们中间参数稍有些不同，后面要慢慢地熟悉使用这些函数。
